## CRUD PHP with Ajax & MySQL

### Stack:
- Php
- Bootstrap 5.3
- Ajax
- MySQL

### How to run: 
- Add this project to the server
- Import city.sql file into the database 
- Start the server

### Basic functionality of the project:
- [C] Adding a new city
- [R] Retrieve list of cities
- [U] Edit selected city
- [D] Delete selected city
- City Search
- Page pagination
